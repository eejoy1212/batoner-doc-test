# 전본서 `purposeText` 파싱 기준

## 대상 원문 예시
- `☐ 외의 용도 (청주지방법원 제천지원 202 3타경6962 [1] 대리입찰)`

## 파싱 목표
- `purposeCourtName`: `청주지방법원 제천지원`
- `caseNumber`: `2023타경6962`
- `itemName`: `1` (원문 `[1]`에서 숫자만 추출)

## 파싱 절차
1. `purposeText` 원문 정리
- 공백 정규화 후 앞뒤 공백 제거.
- 괄호가 있으면 괄호 내부 텍스트를 우선 사용.
- 접두어/노이즈 제거:
  - 체크박스 문자(`☐☑□■`)
  - `외의 용도`, `용도`

2. 사건번호(`caseNumber`) 추출
- 정규식/패턴 기준:
  - `([12]\d)\s*(\d)\s*타\s*경\s*(\d{3,})` (연도가 `202 3`처럼 분리된 경우 허용)
  - `([12]\d{3})\s*타\s*경\s*(\d{3,})`
- 최종 반환 형식:
  - `YYYY타경NNNN...`
  - 예: `202 3타경6962` -> `2023타경6962`

3. 물건번호(`itemName`) 추출
- 우선순위:
  - 대괄호 번호: `\[\s*(\d{1,3})\s*\]`
  - 라벨형 번호: `물건\s*번?\s*호\s*[:：]?\s*(\d{1,3})`
- 예: `[1]` -> `1`

4. 법원명(`purposeCourtName`) 추출
- 원문에서 아래 토큰을 제거한 뒤 남은 텍스트에서 법원명을 탐지:
  - 사건번호 토큰
  - 물건번호 토큰
  - `대리입찰`, `입찰`, `외의 용도`, `용도`
- 최종 법원명 매칭(안정형):
  - `([가-힣\s]*?(?:고등법원|지방법원|법원)(?:\s*(?:[가-힣]+\s*지원|본원))?)`
- 후처리:
  - OCR 띄어쓰기(`청 주 지방법원`)를 `청주지방법원`으로 정규화.

## 로그 확인
- 서버 콘솔에서 아래 로그로 원문/파싱 결과 확인:
- `전본서 ocr 값 :`
